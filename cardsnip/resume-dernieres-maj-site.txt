3. API locale stabilisee
------------------------

L'API FastAPI locale a ete amelioree sans changer son role.

Nom actuel :

CardSnip Local API

Role :

API locale de developpement qui relie :

Next.js -> FastAPI -> SQLite <- scraper Python

Changements recents :

- pagination basique ajoutee sur les endpoints de liste ;
- endpoints 404 propres ajoutes ;
- validation plus claire lors de la creation d'un produit suivi ;
- erreurs API moins opaques.

Endpoints ameliores :

- GET /products?limit=&offset=
- GET /shops?limit=&offset=
- GET /tracked-products?limit=&offset=
- GET /observations?limit=&offset=
- GET /alerts?limit=&offset=

Endpoints de detail ajoutes :

- GET /products/{product_id}
- GET /shops/{shop_id}
- GET /tracked-products/{tracked_product_id}

Exemples de comportement verifie :

- GET /products?limit=1&offset=0 -> 200
- GET /products/999999 -> 404 "Produit introuvable"
- GET /shops/999999 -> 404 "Boutique introuvable"

Fichiers modifies :

- scraper/api.py
- scraper/storage/repositories.py
